1. npx create-react-app my-app --template typescript 
    create-react-app - Название библиотеки.
    my-app - Название приложения.
    --template typescript - Стандартный шаблон настроек( в нашем случае, шаблон TypeScript )
    
2. Игнорировать ругающийся конфигурации TypeScript (это нормально, ts часто обновляется.)

3. Команды (скрипты): 
    start - Запускает ваше приложение.
    build - Собирает приложение в однин большой файл js, css, html.
    test  - Запускает все тесты( тестирование вашего приложения )
    eject - Убирает create-react-app из вашего приложения
    
4. В public удалить все. 

5. В src удалить все, кроме react-app-env.d.ts.

6. Создать index.tsx ( entry-point ) в src.

7. Создать index.html в public который будет главным html файлом (?).

.jsx - Файлы, которое работает с JavaScript.
.tsx - Расширение tsx для работы с TypeScript

8. Постройте приложение ( оно оптимизировано и подходит под прод )

-- Создание тегов

1. Теги создаются функцией, которая возвращает HTML код

Аттрибуты добавляются как обычно к тегам.
    Пример: <button onClick=console.log('hi')>

Перед запуском не забудьте зайти в нужную директорию.
И другие стандартные вещи.
